🔍 Extension hoạt động thế nào?

Ba lớp kiểm tra độc lập — file phải qua cả ba mới được phép

🌐

Lớp 1 — Domain

Kiểm tra domain của trang (upload) hoặc URL tải về (download). Hỗ trợ wildcard *.example.com.

📎

Lớp 2 — Extension file

Kiểm tra đuôi file: .exe, .bat, .ps1... Không phân biệt HOA/thường.

🧠

Lớp 3 — Magika AI

Đọc 4KB đầu của file và dùng mô hình AI để nhận dạng loại thực. File .jpg chứa nội dung EXE vẫn bị phát hiện.

☁️

Remote Policy

Cấu hình blocklist từ server trung tâm. Tự động sync theo chu kỳ — phù hợp triển khai doanh nghiệp.

❓ Câu hỏi thường gặp

Extension có gửi file lên server không?
Không. Magika chạy hoàn toàn trong trình duyệt bằng ONNX WASM. Dữ liệu file không rời khỏi máy tính của bạn.
Tại sao chỉ đọc 4KB đầu file?
Magika chỉ cần header của file để nhận dạng loại — tương tự lệnh file trên Linux. Đọc ít byte hơn giúp không làm chậm trình duyệt, đặc biệt với file lớn.
File bị đổi tên từ .exe thành .jpg có bị phát hiện không?
Có. Đây chính là điểm mạnh của Magika AI — nó đọc nội dung bytes thực của file, không tin vào extension. File virus.jpg chứa mã EXE vẫn bị phân loại là exe và bị chặn.
Extension có ảnh hưởng đến tốc độ upload không?
Rất nhỏ. Việc đọc 4KB và chạy Magika mất khoảng 50–200ms tuỳ thiết bị. Người dùng thường không nhận ra sự chênh lệch.
Nếu server Remote Policy không phản hồi thì sao?
Extension tiếp tục dùng chính sách local đã lưu. Sync thất bại sẽ được ghi log lỗi, không ảnh hưởng đến hoạt động bình thường.
Có thể tắt extension tạm thời không?
Có — click vào icon 🛡️ trên toolbar và gạt toggle Bảo vệ DLP sang Tắt. Extension vẫn chạy nhưng không chặn gì. Bật lại bất kỳ lúc nào.
