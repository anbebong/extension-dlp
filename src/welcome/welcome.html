DLP Shield đã được cài đặt

Ngăn chặn upload/download file theo chính sách — hỗ trợ nhận dạng AI qua Google Magika

⚡ Bắt đầu nhanh

1

Kiểm tra extension đã hoạt động

Click icon 🛡️ trên toolbar → nhấn "Kiểm tra service worker" → phải thấy OK · magikaReady: true. Nếu chưa ready, đợi ~5 giây rồi thử lại (model đang tải).

2

Cấu hình loại file cần chặn

Click "Mở cấu hình" → vào mục Loại file bị chặn → tick chọn định dạng muốn chặn riêng cho Upload và Download → nhấn Lưu cấu hình.

3

Thêm domain được phép (tuỳ chọn)

Vào mục Domain được phép → thêm domain nội bộ (vd: *.company.com) → các domain này sẽ bỏ qua mọi kiểm tra file.

🔍 Cách hoạt động

⬆️

Chặn Upload

Đọc bytes đầu file khi người dùng chọn file qua input[type=file], gửi sang Magika AI để nhận dạng, chặn nếu vi phạm policy.

⬇️

Chặn Download

Bắt sự kiện download sau khi Chrome nhận được tên file thật từ server, kiểm tra extension và hủy download nếu vi phạm.

🧠

Magika AI

Nhận dạng file theo nội dung bytes — không phụ thuộc đuôi tên. File virus.jpg chứa EXE bytes vẫn bị phát hiện.

☁️

Remote Policy

Cấu hình blocklist từ server trung tâm, tự động sync theo chu kỳ — phù hợp triển khai doanh nghiệp.

❓ Câu hỏi thường gặp

Extension có gửi file lên server không?
Không. Magika chạy hoàn toàn trong trình duyệt. Dữ liệu file không rời khỏi máy.
Tắt extension tạm thời bằng cách nào?
Click icon 🛡️ → gạt toggle Bảo vệ DLP sang Tắt. Bật lại bất kỳ lúc nào.
Policy thay đổi nhưng không áp dụng?
Nhấn Lưu cấu hình sau mỗi thay đổi. Policy được áp dụng ngay lập tức sau khi lưu.

Sẵn sàng cấu hình?

Mở trang Options để chọn định dạng file cần chặn
